# SegFormer3D Project Website

This directory contains the project website for **SegFormer3D: An Efficient Transformer for 3D Medical Image Segmentation**.

## View the Website

The website is hosted via GitHub Pages. To enable it:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Set source to "Deploy from a branch"
4. Select `main` branch and `/docs` folder
5. Click "Save"

The website will be available at: `https://yourusername.github.io/SegFormer3D/`

## Local Preview

To preview the website locally, simply open `index.html` in your web browser.

## Structure

- `index.html` - Main website content
- `style.css` - Styling and layout
- The website references GIF visualizations from `../visualizations/`

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Qualitative Results**: Showcases GIF comparisons across all three datasets
- **Quantitative Results**: Performance tables for BRaTS, Synapse, and ACDC
- **Method Overview**: Brief explanation of key components
- **Citation**: BibTeX entry for easy citation

## Customization

To update the website:
- Edit `index.html` for content changes
- Edit `style.css` for styling modifications
- Update links in the header section to point to your paper/code
